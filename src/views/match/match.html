¿Como usar?

Para contabilizar puntos en el sistema, siga estos pasos:

  1. Pulse el botón "Activar micrófono" ubicado en la parte superior de la pantalla. Esta acción permitirá el uso del micrófono para registrar los puntos.
  2. Para sumar un punto a un equipo, mencione la palabra "Punto" seguida del nombre del equipo correspondiente. Por ejemplo:
  • "Punto rojo" para asignar un punto al equipo rojo.
  • "Punto azul" para asignar un punto al equipo azul.

Esta funcionalidad permite registrar puntos de manera sencilla mediante el uso de comandos de voz. Asegúrese de que su pronunciación sea clara para garantizar el reconocimiento adecuado del sistema.

Como alternativa también puede apretar el boton 'Contar punto'
